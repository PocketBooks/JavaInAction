# Modern-Java-In-Action

### 매주 스터디 방법

1. 매주 정해진 발표자가 정리한 내용을 발표한다.
   - 개념부분 위주, 추가로 예제소개까지해도 상관없음
2. 각자 정리된 내용을 pr을 날린다

  - 문서 형식 혹은 코드등을 정리한다.

  - 예제코드는 책의 예제를 아니면 본인이 만들거나 검색해서 긁어와도 됌

3. 발표자의 발표자료로 뼈대를 구축하고 발표를 통해 피드백 내용 추가될 내용들을 추가하여 Git에 추가한다.
